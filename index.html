1. Install Aplikasi Telegram

Berikut adalah cara install telegram :

- Link aplikasi telegram : https://play.google.com/store/apps/details?id=org.telegram.messenger

- Download telegram kemudian install

- Daftarkan nomor hp (seperti daftar whatsapp)

- Verifikasi Kode OTP dari Sms yang masuk

2. Install Aplikasi Vtube

Berikut adalah cara install Vtube :

- Link aplikasi Vtube : https://play.google.com/store/apps/details?id=co.vtube.fvtech

- Tekan install dan tunggu sampai selesai

1. Registrasi OTP Vtube

Berikut adalah cara Registrasi OTP Vtube :

- Buka salah satu link bot ini :
https://t.me/VtubeOTPBot
https://t.me/Vtube_OTP_Bot
https://t.me/Vtube_OTP_2_Bot
https://t.me/Vtube_OTP_3_Bot

- Kemudian Klik mulai / start

- Kemudian tekan Register

- Kemudian tekan Send Us Your Contact

- Terakhir pilih Bagi Kontak / Share my contact

2. Registrasi akun Vtube

Berikut adalah cara Registrasi akun Vtube :

- Buka Aplikasi Vtube yang sudah diinstall tadi

- klik Skip

- klik Me/saya(*Dipojok kanan bawah)

- klik Konfirmasi

- klik Registrasi(*Dipojok kanan atas)

- Isikan Nomor telepon hp telegram(tanpa 0) , misal nomor saya 088787879827 maka masukkan 88787879827(tanpa 0)

- Lalu buat kata sandi atau password serta konfirmasi sandi

- Kemudian klikRequest OTP/Minta OTP

- Kemudian buka telegram anda dan masukkan kode otp yang ada di telegram ke kolom OTP di aplikasi vtube

- Kemudian masukkan kode captcha sesuai gambar

- Lalu klik tulisan kecil dibawah captcha TERM & CONDITION /SYARAT & KETENTUAN

- Kemudai geser kebawah didalam popup, kemudian tekan tombol X

- Terakhir klik Submit dan akan muncul tulisan Registration Succesfull

3. Login Akun Vtube

Berikut adalah cara Login akun Vtube :

- Pertama Klik tombol kembali

- klik Login Option

- klik Kata Sandi

- Kemudian masukkan nomor yang didaftarkan diatas (Tanpa angka 0) serta masukkan kata sandinya

- Isi kode captcha

- Terakhir klik Masuk

1. Verifikasi Akun Vtube

Berikut adalah cara Verifikasi akun Vtube :

- Pertama KlikMe/Saya

- Isikan Nomor whatsapp atau Telegram

- Masukkan Age / Umur

- Masukkan City / Kota (*Tempat tinggal anda)

- Selanjutnya Klik Confirm

- Kemudian geser kebawah dan pilih kotak bertuliskan Verifikasi

- Selanjutnya isi nama sesuai dengan KTP anda

- Isi NIK sesuai dengan KTP anda

- Selanjutnya Masukkan REFFERAL CODE : 2480468

- Kemudian klik Kirimkan/Submit
